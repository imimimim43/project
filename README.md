## 포드-풀커슨 알고리즘(Ford-Fulkerson algorithm)

<br>


■ 포드-풀커슨(Ford-Fulkerson)

- 유량 네트워크에서 소스와 싱크 사이에 흐를 수 있는 최대 유량(Maximum Flow)을 구하는 알고리즘이다.

<br>

■ 유량 네트워크(flow network)

- 각 간선에 용량이라는 추가 속성이 존재하는 방향 그래프이다.
